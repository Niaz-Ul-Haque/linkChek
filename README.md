# linkChek

CLI tool to check links from a html file or url

### Install

```
npm install -g <https://github.com/joelazwar/linkChek>
```

### How to use

```
- linkChek -l [Filename or Webpage URL]
```

![GIF PREVIEW](assets/htmlpreview.gif)
